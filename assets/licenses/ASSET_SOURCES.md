# Asset Sources

Record every external asset here **before** committing it. Assets without a recorded, compatible license must not be committed.

## Pipeline

| Stage | Folder | Meaning |
|-------|--------|---------|
| Source | `assets/source/` | Editable originals (`.blend`, source textures) |
| Raw | `assets/raw/` | Downloaded, unprocessed third-party assets |
| Processed | `assets/processed/` | Game-ready exports (`.glb` / `.gltf`) |

## Rules

- Only commit assets whose license permits use and redistribution in this project.
- Record the source, license, and author for every external asset below.
- Keep original license files alongside downloads where the license requires it.
- When in doubt, do not commit — ask first.
