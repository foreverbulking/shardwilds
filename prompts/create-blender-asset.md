# Create Blender Asset Prompt

Read:

- `docs/06-art-direction.md`
- `docs/11-asset-pipeline.md`

Task:

Create or modify one asset only.

Rules:

- Use Shardwilds naming prefix `sw_`.
- Keep low-to-mid poly.
- Use stylized colors.
- Keep silhouette readable.
- Export as `.glb` when complete.
- Do not overwrite source files.
- Update asset manifest or license docs if needed.

Output:

- Files created
- Export path
- Scale notes
- License/source notes
