# Art Direction

## Style Summary

Shardwilds uses a stylized anime fantasy look optimized for browsers.

The goal is:

- Charming
- Colorful
- Readable
- Lightweight
- Animation-friendly
- Original

## Visual Traits

Use:

- Bright colors
- Large readable silhouettes
- Cute heroic characters
- Slightly chibi proportions
- Oversized weapons
- Expressive animations
- Colorful fantasy environments
- Low-to-mid poly assets
- Toon or semi-toon materials
- Simple flashy VFX
- Clear skill effects
- Classic MMO UI readability

Avoid:

- Realistic PBR as the main style
- High-poly characters
- Dense foliage
- Heavy post-processing
- Photoreal textures
- Direct copies of reference game designs

## First Zone: Hearthvale

Hearthvale should include:

- Cozy village
- Green fields
- Fishing pond
- Small mine entrance
- Crafting table
- Trade post
- Curved-roof houses
- Soft fantasy lighting
- Friendly starter creatures

## Asset Sources

Preferred free sources:

- Quaternius
- Kenney
- Selected itch.io packs with clear licenses
- OpenGameArt with license review

Preferred licenses:

- CC0
- Public domain
- MIT
- Apache-2.0

Avoid:

- Personal use only
- Editorial only
- Unclear licenses
- GPL assets
- CC-BY-SA unless explicitly approved

## Naming

Use project prefixes:

```txt
sw_tree_01.glb
sw_copper_node_01.glb
sw_trade_pack_01.glb
sw_house_hearthvale_01.glb
```
