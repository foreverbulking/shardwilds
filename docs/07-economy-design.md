# Economy Design

## Economy Philosophy

The economy should make the world matter.

Players should gather, refine, craft, transport, sell, and risk goods across regions.

The economy must be designed with bots and dupes in mind from day one.

## Core Loops

### Gathering Loop

```txt
Find node -> gather resource -> store in inventory -> refine/craft/use/sell
```

### Crafting Loop

```txt
Gather materials -> use station -> craft item -> consume/use/trade
```

### Trade Pack Loop

```txt
Gather materials -> craft pack -> carry through world -> deliver -> earn reward
```

## Gold Rules

Gold creation must be logged.

Gold faucets:

- Trade pack delivery
- Quest rewards later
- Dungeon completion later

Gold sinks:

- Repairs later
- Crafting fees later
- Housing upkeep later
- Travel fees later

## Anti-Bot Economy Rules

Avoid:

- Uncapped gold from fishing
- Instant crafting profit loops
- Client-created item results
- Rare drops from endlessly farmable zero-risk mobs

Use:

- Server-side ledgers
- Rate limits
- Variable respawn timers
- Regional demand
- Risk/reward routes
